h2. Socialite

A Ruby on Rails plugin that provides an easy way to share your content on the most popular social networking sites.

h3. What it does

Socialite simply adds a view helper @socialite@ to your application. Pass it a URL, content, and title and it will generate links to submit your content to the most popular social networking sites.

h3. Requirements

None, yo.

h3. How to Install

Install the Socialite plugin:
<pre>./script/plugin install git@github.com:/mdarby/socialite.git</pre>

h3. How to Use

<pre>
  <%= socialite(:url => URL, :title => TITLE, :content => CONTENT) %>
</pre>
